# Apartment-sales-information-website
## 아이템 배경
* 아파트 매물 정보 검색의 필요성
  * 고객들이 시, 구, 동 별로 실시간 아파트 매매 정보를 검색할 수 있으며, 아파트 주변 상권 정보까지 확인 가능.
